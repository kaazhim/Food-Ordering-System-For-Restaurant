1.1 PROJECT BACKGROUND 

Burger’s House is a burger restaurant that wants to make its ordering system more organized. This is stated because they are still using the traditional way of taking orders, where the employee will write the order on paper. This method will cause a few inconveniences, such as the employee making some errors while writing down the orders, bad handwriting, and the order paper might be lost if the employee is clumsy. Thus, customers will receive the wrong order and receipt. Other than that, it is also hard to count the restaurant’s profit for a day if they do it manually. Nowadays, it is difficult to start a new small-scale business while trying to compete with other well-established company. In the era where technology is growing at a fast-paced time, Burger’s House needs to have a food ordering system to make their business more efficient 
and organized. 

1.2 OBJECTIVES 

1. To develop an organized food ordering system. 
By having a food ordering system, it will be easier for the restaurant’s staff to get the 
information about their daily sales report. The system also displays the food menu and 
receipt, which will be useful for the restaurant to use it when needed. 

2. To ensure that the total price of an order is correct. 
This will avoid miscalculation of an order because it is more accurate since the system 
will count the total price automatically. 

3. To design a user-friendly system. 
The user interface of the system is clear and attractive to the user. This will make it easy 
to understand because the system is easy to use when it is user-friendly.


1.3 DESCRIPTION 

This food ordering system will display three choices in the main menu for the 
user, which are the food menu, the daily report sales for the restaurant in a day, and a choice to exit the program. Firstly, if the user chooses “Make an order” by entering ‘1’, they have 
to enter their name, and then will be welcomed and asked to select the menu that they 
would like to order. There are a lot of foods that they can choose from Burger’s House’s 
menu. The menu consists of burgers, drinks, and side dishes. The user just needs to pick 
the menu type that they want to choose, and the system will bring the user to their 
preferences. If the user enters the wrong input, they will be asked to redo it again. They 
can also press ‘y’ if they want to make another order. Then, after the user is satisfied 
with their order, they will press ‘n’ indicating that they are done ordering. The system 
will display the total price and the user will put the amount of payment that they will 
make. If the user inputs less money than the total price of their order, they need to insert 
the total payment again. Next, the system will generate a receipt where it will count the 
subtotal, total tax, total after tax, cash tendered, and change. Other than that, it will also 
display the items that the user ordered. The user then can enter any input to go back to 
the main menu.  

Secondly, the user can also look at their daily report sales by entering ‘2’ at the 
main menu where the program will calculate the total sales in a day for Burger’s House. 
In the daily report sales, it will count the total amount of customers, grand total, tax total, 
and total profit. By this, it will be easier for the user to keep track of their business 
performance. To go back to the main menu, the user will need to enter ‘0’. Lastly, if the 
restaurant is done for the day, they can enter ‘3’ to exit the program where the system 
will reset the data for that day and get ready for the next day. 
This user-friendly ordering system will help Burger’s House a lot because of the 
amazing system. With this, they can attract more customers to go to their restaurant since 
their service is excellent, fast, and accurate. Thus, they will no longer face problems like 
before where they will a lot of complaints about their ordering system and slow services.
